# JM Dushi Treats 🍪

Welkom bij de website van **JM Dushi Treats** - vers gebakken koekjes met een vleugje Curaçao!

## 🌟 Wat Is Dit?

JM Dushi Treats is een koekjesbedrijf dat heerlijke, handgemaakte koekjes bakt. Deze website laat onze menu, reviews van klanten, en besteloptie zien.

## 🚀 Functies

- **Responsief Design** - Mooi op alle apparaten (telefoon, tablet, computer)
- **Menu** - 8 verschillende koekjessoorten
- **Klanten Reviews** - Echte feedback van gelukkige klanten
- **WhatsApp Bestellen** - Direct via WhatsApp bestellen
- **Moderne UI** - Schone en aantrekkelijke interface

## 📂 Bestanden

- `index.html` - Hoofd website (alles in één bestand)
- `logo.png` - JM Dushi Treats logo
